# CHANJUN KWON Portfolio Website

노션 포트폴리오를 기반으로 제작한 단독 웹사이트 버전입니다.

## 실행

```bash
cd "/Users/chriskkk/Documents/New project"
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 열기.

## 구성 파일

- `index.html`: 페이지 구조와 콘텐츠
- `styles.css`: 전체 디자인/반응형 스타일
- `script.js`: 스크롤 리빌 애니메이션, 프로젝트 필터, 프로젝트/자기소개서 랜딩 뷰어
- `project-details.js`: 노션 프로젝트 상세 데이터(이미지/요약)
- `intro-details.js`: 노션 자기소개서 상세 데이터
